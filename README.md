# 👁️ Relax Eye – Eye Wellness Web Application

Relax Eye is a web-based eye wellness application developed as an academic project. It provides users with essential eye care resources, including eye exercises, blue light awareness, vision health tips, eye testing tools, and wellness reminders through a simple and interactive interface.

## 🌟 Features

- Secure login interface
- User-friendly dashboard
- Eye exercise reminders
- Blue light awareness
- Vision health tips
- Screen time tracking
- Personalized eye care plan
- Eye testing tools
- Relaxing audio guides
- Night mode guidance
- Eye health checklist
- Simple and responsive interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
